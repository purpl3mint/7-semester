Задача аппроксимации объекта с дискретным выходом схематически показана на рис. 3.2, где подчеркивается, что входы объекта могут задаваться тремя способами: а) числом, б) 2-лингвистическим термом, в) 3-по принципу термометра.

Идея метода, предлагаемого ниже для решения этой задачи состоит в использовании нечетких логических уравнений. Эти уравнения строятся на базе матрицы знаний или изоморфной ей системы логических высказываний (3.10) и позволяют вычислять значения функций принадлежности различных решений при фиксированных значениях входных переменных объекта. В качестве искомого решения выбирается решение с наибольшим значением функции принадлежности.

Рубрика MATLAB & Toolboxes
  Рубрика
MATLAB & Toolboxes состоит из разделов, посвященных MATLAB, MATLAB Toolboxes, Simulink, Simulink Blocksets. У разделов есть ведущие, материалы в этих разделах - авторские.

Ведущие разделов:
- Simulink - д.т.н., доцент И.В.Черных
- Optimization - д.т.н. А.Г.Трифонов
- Genetic Algorithm and Direct Search Toolbox - ведущий д.т.н. А.Г.Трифонов
- Statistics Toolbox - к.т.н., доцент З.В.Мищенко
- SNAE Toolbox - к.т.н., доцент В.Г.Потемкин
- Control System Toolbox - к.т.н., доцент Е.В.Никульчев
- Оптимальные и робастные системы управления (Mu-Analysis and Synthesis Toolbox, Robust Control Toolbox, LMI Control Toolbox, Model Predictive Control Toolbox) - кафедра компьютерных технологий и систем факультета прикладной математики - процессов управления СПбГУ
- System Identification Toolbox - к.т.н., доцент Е.В.Никульчев
- Fuzzy Logic Toolbox - к.т.н., доцент С.Д.Штовба
- Signal Processing Toolbox - к.т.н., доцент А.Б.Сергиенко
- Image Processing Toolbox - к.т.н. И.М.Журавель
- Image Acquisition Toolbox - к.т.н. И.М.Журавель
- Wavelet Toolbox - ведущий к.т.н., доцент К.А.Алексеев
- Communications Toolbox - к.т.н., доцент А.Б.Сергиенко
- Stateflow - к.т.н., доцент Г.Н.Рогачев
- SimPowerSystems - д.т.н., доцент И.В.Черных
- Fixed-Point Blockset - к.т.н., доцент К.Г.Жуков
- Nonlinear Control Design Blockset - кафедра компьютерных технологий и систем факультета прикладной математики - процессов управления СПбГУ
- н! Приложения с GUI и дескрипторная графика - к.ф.-м.н., доцент И.Е.Ануфриев
- н! Интерполяция и приближение данных в MATLAB - канд. физ.-мат. наук, доцент кафедры "Прикладная математика" И.Е.Ануфриев
- н! Financial Derivatives Toolbox - к.т.н. Силантьев Сергей Алексеевич.
